# Calculadora Simples

Calculadora em Python com interface no terminal. Suporta operações matemáticas básicas e avançadas.

## Operações disponíveis

- **1.** Adição (+)
- **2.** Subtração (-)
- **3.** Multiplicação (*)
- **4.** Divisão (/)
- **5.** Divisão inteira (//)
- **6.** Resto/Módulo (%)
- **7.** Potência (**)
- **8.** Raiz quadrada (√)

## Como executar

```bash
python calculadora.py
```

## Requisitos

- Python 3.6 ou superior (usa apenas biblioteca padrão)
